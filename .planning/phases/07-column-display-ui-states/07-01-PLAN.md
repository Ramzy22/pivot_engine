Implement the state-management foundation for Phase 7 by making visibility and sizing first-class controlled UI state with durable behavior across refresh flows.

Purpose: UI-03 and UI-04 are reliability requirements. This plan hardens ownership of these states and removes ambiguity in refresh behavior.
